# WindowsOpenBSDdistributor

Загрузите репозитории gladiola на USB-накопитель в Windows, а затем установите их на машину с OpenBSD — даже если система OpenBSD не имеет прямого доступа к Интернету.

Оба скрипта поддерживают **интерактивный выбор**, позволяя точно указать, какие репозитории загружать или устанавливать каждый раз.

---

## Шаг 1 – Загрузка на USB-накопитель (Windows)

### Предварительные требования

* Установлен [Git для Windows](https://git-scm.com/download/win) и добавлен в `PATH`.
* Подключён USB-накопитель (например, диск `E:`).

### Запуск скрипта PowerShell

```powershell
# Разрешить выполнение локальных скриптов (один раз, при необходимости запустить от имени Администратора)
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

| Режим | Команда |
|---|---|
| **Интерактивный** – окно выбора GUI | `.\windows\download_repos.ps1 -DriveLetter E` |
| **Конкретные репозитории** – без GUI | `.\windows\download_repos.ps1 -DriveLetter E -Repos "OBJC-codespaces","OpenBSDHomemadeBlockScripts"` |
| **Все репозитории** – без GUI | `.\windows\download_repos.ps1 -DriveLetter E -All` |
| С токеном GitHub (повышает лимит запросов) | добавить `-Token ghp_yourToken` к любой команде выше |

**Интерактивный режим** получает все публичные репозитории gladiola через GitHub API и открывает окно `Out-GridView` со списком имён репозиториев. Удерживайте **Ctrl** или **Shift** для выбора нескольких репозиториев, затем нажмите **ОК**.

Скрипт клонирует каждый выбранный репозиторий в `<БукваДиска>:\gladiola_repos\`. Если репозиторий уже был клонирован ранее, он обновляется и сбрасывается до последнего `HEAD`.

Безопасно извлеките USB-накопитель после того, как скрипт сообщит об успехе.

## Шаг 2 – Установка на OpenBSD

### Предварительные требования

* Доступ root (или `doas`) на машине с OpenBSD.
* USB-накопитель физически подключён к машине с OpenBSD.

### Монтирование USB-накопителя

```sh
# Найти имя устройства (искать USB-накопитель, часто sd1 или sd2)
sysctl hw.disknames

# Смонтировать (заменить sd1i реальным разделом)
doas mount -t msdos /dev/sd1i /mnt/usb
```

### Запуск скрипта установки

| Режим | Команда |
|---|---|
| **Интерактивный** – нумерованное меню | `doas sh openbsd/install_repos.sh` |
| **Конкретные репозитории** – без меню | `doas sh openbsd/install_repos.sh -r OBJC-codespaces,OpenBSDHomemadeBlockScripts` |
| **Все репозитории** – без меню | `doas sh openbsd/install_repos.sh -a` |
| Пользовательский источник / назначение | добавить `-s /mnt/usbkey -d /home/myuser/gladiola` |

**Интерактивный режим** перечисляет каждый репозиторий из `gladiola_repos/` на USB-накопителе с номером, затем предлагает ввести нужные номера (например, `1 3 5`). Нажмите **Enter** без ввода, чтобы установить всё.

Для каждого выбранного репозитория скрипт:
1. Копирует его в `<каталог_установки>/` (по умолчанию: `/usr/local/gladiola`)
2. Устанавливает `chmod 755` для всех файлов `.sh`
3. Выполняет `make install`, если присутствует `Makefile` (вывод отображается только при ошибке)

### Размонтирование USB-накопителя после завершения

```sh
doas umount /mnt/usb
```
